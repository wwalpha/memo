# AWS Toolkit for Visual Studio Code

## Prepre
- Git
- Node.js
- Python

## Installation

```s
$ pip install aws-sam-cli --upgrade
$ git clone https://github.com/aws/aws-toolkit-vscode.git
$ cd aws-toolkit-vscode
$ npm install
$ npm run package
$ code --install-extension aws-toolkit-vscode-<VERSION>.vsix
```
